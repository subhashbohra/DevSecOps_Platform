# STAR Story Template

Copy this for any new story. Keep it ~2 minutes spoken.

```
TITLE: ______
Tags (LPs / Googleyness / Atlassian value): ______

SITUATION (10-20s): ______
TASK (10-20s): ______
ACTION (40-60s, mostly "I", credit the team): ______
RESULT (20s, QUANTIFIED): ______
LEARNING / what I'd do differently (10s): ______

Scope-framing line (elevates to Principal/L6): ______

3 likely follow-ups + my answers:
1. ______
2. ______
3. ______
```

Checklist:
[ ] Under 2 minutes
[ ] Real, defensible numbers
[ ] Led with scope, landed quantified result
[ ] Ends with a genuine lesson
[ ] Shared credit, no ego framing
